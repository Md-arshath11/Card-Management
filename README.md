# Card-Management

# Dashboard
- Displays user cards in a horizontally scrollable   
- Frequently used actions (Recharge, Bill Payment, Send Money, etc.)  
- Services section with Vertically scrollable

# All Cards Screen
- Shows all cards added by user  
- Cards dynamically update when a new card is added  

# Add Card
- Add new cards with:
  - Bank name  
  - Card number  
  - Card type (Visa / Mastercard)  
  - Expiry date  
- Added cards automatically reflect in Dashboard & All Cards screens

# Card Details Screen
- Shows fake chart  
- Displays current balance  
- Scrollable transaction history

- Used Datastore to store add new card
